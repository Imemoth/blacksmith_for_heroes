# MVP Scope

## MVP cél

A cél nem a teljes RPG blacksmith játék, hanem a core idle-crafting loop validálása.

## MVP-ben benne van

| Rendszer | Státusz |
|---|---|
| Iron Ore / Wood resource tick | Must Have |
| Gold | Must Have |
| Reputation XP | Must Have |
| Forge Sigil | Light prestige után |
| Sword / Bow / Staff / Axe | Must Have |
| Item level | Must Have |
| Rarity | Must Have |
| Affix | Must Have |
| Power és sell value | Must Have |
| Inventory | 20 slot induláskor |
| Market fallback | 35% value, 0 Rep |
| Guild Contracts | Gold fókusz |
| Hero Commissions | Reputation fókusz |
| Blueprint Shop | Rep unlock + Gold purchase |
| Workshop upgrades | Must Have |
| Tier 1–3 | Must Have |
| Advanced blueprints | Tier 2 mögött |
| Event Log | Must Have |
| Hero feedback | Must Have |
| Masterwork | Must Have |
| Light prestige | Must Have |

## Fő MVP loop

1. Resource termelődik.
2. Player craftot indít.
3. Item roll lefut.
4. Item inventoryba kerül.
5. Player leadja:
   - Guild Contractra,
   - Hero Commissionre,
   - vagy Market fallbackre.
6. Reward érkezik:
   - Gold,
   - Reputation,
   - esetleg feedback.
7. Player blueprintet vagy upgrade-et vesz.
8. Jobb orderök és jobb craft capek nyílnak.
9. Rep 5 + Tier 3 után Masterwork és prestige.

## Végső MVP milestone

Az első teljes run célja:

- Rep Level 5,
- Tier 3,
- Masterwork Frame owned,
- Epic vagy jobb Lv 15+ item,
- 5+ hero commission,
- 5+ guild contract,
- első Forge Legacy prestige.
